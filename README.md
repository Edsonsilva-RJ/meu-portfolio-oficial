## 🛠️ Tecnologias Utilizadas
* **React.js** (Frontend)
* **Vite** (Build Tool & Servidor de Desenvolvimento)
* **CSS3** (Estilização com foco em Responsividade e Glassmorphism)
* **JavaScript (ES6+)**

## 📐 Funcionalidades
- [x] **Navbar Responsiva:** Menu hambúrguer para dispositivos móveis.
- [x] **Seção Sobre:** Detalhes da minha formação e transição de carreira.
- [x] **Galeria de Projetos:** Cards integrados com repositórios do GitHub.
- [x] **Download de CV:** Link direto para o currículo em PDF.
- [x] **Contato:** Integração com LinkedIn e E-mail.

## ⚙️ Como rodar o projeto localmente
```bash
# Clone o repositório
git clone [https://github.com/SEU_USUARIO/portfolio-final.git](https://github.com/SEU_USUARIO/portfolio-final.git)

# Entre na pasta
cd meu-portfolio

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
